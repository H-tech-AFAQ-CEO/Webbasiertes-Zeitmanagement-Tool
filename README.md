# ⏱️ TimeTrack Pro

A lightweight, zero-configuration **web-based time tracking system** built with PHP, SQLite, and PHPMailer. Designed for simplicity, speed, and easy deployment — no MySQL required.

---

## 🚀 Quick Start

Open in your browser:
http://localhost/Webbasiertes%20Zeitmanagement-Tool/

✅ No installation required — works instantly!

---

## ⚙️ Configuration (Optional)

```env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
SMTP_FROM=your-email@gmail.com

ADMIN_PASSWORD=your-secure-password
